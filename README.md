# Python Arcade Game
Console based arcade with multiple games and MySQL leaderboard.

## Games Included
Wordle
Quiz
Go Fish
Typing Game
Hand Cricket
Rock Paper Scissors
Chopsticks

## Requirements
Python 3
MySQL

## Setup Instructions
Step 1. Install dependency

pip install -r requirements.txt

Step 2. Setup database

Open MySQL

Run database.sql file

Step 3. Change password

Open arcade.py

Change:

password="your_password"

Step 4. Run program

python arcade.py

## Features
Leaderboard
Score storage
Multiple games
Menu driven
